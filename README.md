# To-Do List (Tkinter GUI)

## CodTech Internship Task

Intern ID: CITS2549

## Project Description

To-Do List (Tkinter GUI) is a Python-based desktop application that helps users manage their daily tasks through a simple graphical user interface. Users can add new tasks, delete selected tasks, and clear all tasks with confirmation prompts. The application is built using Python's built-in Tkinter library and demonstrates the basics of GUI development and event handling.

## Technologies Used

- Python
- Tkinter

## Features

- Add new tasks
- Delete selected tasks
- Clear all tasks
- Error message for empty task input
- Confirmation dialog before clearing all tasks
- Simple and user-friendly GUI

## How to Run

1. Make sure Python 3 is installed.

2. Run the application:

   ```bash
   python main.py
   ```
